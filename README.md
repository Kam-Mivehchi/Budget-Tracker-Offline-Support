# Budget-Tracker-Offline-Support


## Description
 - Progressive Web Application
 - Budget Tracker with Animated Graph
 - Allows client to manage spending by adding or subtracting transactions
 - Features 
    - lazy load
    - service worker
    - offline support
 - Deployed on heroku


# Deployment Link
    - https://budget-tracker-222.herokuapp.com/
